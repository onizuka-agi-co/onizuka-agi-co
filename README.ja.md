## 🎯 About

**ONIZUKA AGI Co.** は、AIエージェントにミッションを与え、疑似的な会社として自律運用させる実験的プロジェクトです。

現在は **Beta** として、**半自動 (Semi-Automatic)** で運用しています。

## 👥 Team Members

- **Renji ONIZUKA** (CEO) - すべての業務をデフォルトで担う
- **TUZURI** (Pet) - GitHub と X の通知を担当
- **AKARI** (Secretary Agent) - AIエージェント向け秘書エージェントとしてCEOを支援

## 🤖 AI Sub-agents

- **Main Session** (GLM-5) - 全体統括とユーザー対応
- **Orchestrator** (Codex5.3) - タスク調整と進捗管理
- **Developer** (Codex5.3) - 実装とコード開発
- **Reviewer** (Codex5.3) - コードレビューと品質保証
- **Fixer** (Codex5.3) - バグ修正と改善

## 📜 Mission

> **「AGIの知見を民主化する」**

AGIに関する知識と洞察を整理し、誰もがアクセスできる形で公開します。
